# Dados de referência (raças, antecedentes, classes)

Extraídos de `5etools-mirror-3/5etools-src` (repositório público de dados, não é código do Plutonium) em 2026-07. Cada arquivo contém só nome, fonte e o que o item concede (perícias, idiomas, equipamento inicial) — sem texto de regra completo, pra manter o bundle do site pequeno.

Conteúdo de fã compilado a partir de livros oficiais — mesma zona cinzenta de licenciamento de qualquer ferramenta desse tipo (Plutonium incluso), aceitável pra uso privado da campanha.

Pra atualizar: rodar de novo o script de extração (não versionado, mas documentado no histórico da conversa que criou isso) contra os arquivos `data/races.json`, `data/backgrounds.json`, `data/class/class-*.json` do repositório acima.
